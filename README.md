# Демонстраційний проєкт «TripMate» 

## Перша версія: Лабораторна робота 2 (HTML)

Це статичний HTML-макет для вигаданого сервісу **TripMate**, який пропонує готові тури до складних для самостійної подорожі країн.

Проєкт створено в рамках лабораторної роботи з предмету "Веб-розробка". Мета — відпрацювати навички створення семантичної HTML-розмітки, організації файлової структури та навігації між сторінками.

**Використані технології:**
* Тільки чистий `HTML`
* *CSS та JavaScript на даному етапі не використовувалися.*

## Оновлення: Лабораторна робота №3 (CSS)

У цій лабораторній роботі було додано оформлення та адаптивність:

* **CSS3:** Використано каскадні таблиці стилів для оформлення всіх сторінок.
* **Flexbox:** Використано для створення гнучкої сітки карток турів та вирівнювання елементів навігації.
* **Адаптивний дизайн (Media Queries):** Сайт адаптовано для мобільних пристроїв (ширина екрану до 768px). Реалізовано **Бургер-меню** без використання JavaScript.
* **Модульна структура:** Стилі розділено на окремі файли (`base.css`, `layout.css`, `components.css`) та об'єднано через `@import`.
* **Валідація форм:** Додано візуальну індикацію (зелений/червоний колір) для перевірки коректності введених даних.

## Оновлення: Лабораторна робота №4 (JavaScript)

Додано інтерактивність та логіку на стороні клієнта (Client-Side Logic):

* **DOM Manipulation:**
  * Реалізовано зміну стилів елементів (заголовки карток) через JS.
  * Динамічне додавання нових елементів (повідомлення в footer) методами `createElement` та `append`.
  * Реалізовано акордеон "Читати більше" (показ/приховування тексту).

* **Події (Events):**
  * **Click:** Перемикання темної/світлої теми сайту.
  * **Keyboard (keydown):** Зміна розміру шрифту клавішами `ArrowUp` / `ArrowDown`.

* **Робота з формами (Forms & Validation):**
  * Відключено стандартну відправку форми (`preventDefault`).
  * Реалізовано власну валідацію: перевірка довжини імені, наявності `@` в email, довжини повідомлення.
  * Візуалізація помилок (червона рамка та текст під полем).

* **Browser Storage (LocalStorage):**
  * Реалізовано збереження обраної теми (Dark/Light). При перезавантаженні сторінки вибір користувача зберігається.

## Структура проєкту

Проєкт має наступну файлову структуру:

```text

├── images/ # Папка для зображень (лого, фото турів) 
├── css/
|    ├── base.css
|    ├── components.css
|    ├── layout.css
|    └── style.css
├── js/ 
|    └── main.js
| 
├── index.html # Головна сторінка 
├── catalog.html # Каталог турів 
├── tour-details.html # (Шаблон) Детальна сторінка туру 
├── blog.html # Сторінка блогу 
├── about.html # Сторінка "Про нас" 
├── contacts.html # Сторінка контактів (з формою) 
├── login.html # Сторінка входу (з формою) 
├── cart.html # Сторінка кошика 
| 
└── README.md # Цей файл з описом проєкту

```
